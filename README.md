# 🚀 Mxnet AI

**Herramienta de prueba de red con IA + Nmap**  
*Multiplataforma: Linux | Windows*

![Versión](https://img.shields.io/badge/version-1.0-blue)
![Python](https://img.shields.io/badge/python-3.8%2B-green)
![Nmap](https://img.shields.io/badge/nmap-7.80%2B-red)

## 🧠 Características
- Escaneo inteligente de puertos con **Nmap**
- **IA** para recomendaciones de seguridad
- Reportes automáticos en TXT/HTML
- Soporte para **Linux** y **Windows**

## 📦 Instalación
```bash
git clone https://github.com/Falconmx1/Mxnet-AI.git
cd Mxnet-AI
pip install -r requirements.txt

🎯 Uso básico
python mxnet.py --target 192.168.1.1 --scan-type full
